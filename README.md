
Nombre del proyecto: 
AnimeGR--Tienda Online de Productos Anime

Descripción:
AnimeGR es una tienda web diseñada para ofrecer productos relacionados con el anime, como figuras, ropa y mangas.
El proyecto incluye varias secciones como inicio, productos, categorías, contacto, registro e inicio de sesión.

Tecnologías usadas: 
HTML – estructura y organización de las páginas.
CSS – estilos, diseño visual, colores, botones y maquetación responsiva.
Git & GitHub – control de versiones y repositorio del proyecto.
Imágenes y SVGs de Tabler Icons – para mejorar la presentación visual.

Instrucciones de instalación o visualización:  
1. Clonar o descargar el repositorio:
git clone <URL-del-repositorio>
2. Abrir la carpeta del proyecto.
3. Navegar a la ruta principal donde se encuentra index.html.
Abrir el archivo en cualquier navegador:
Doble clic en index.html
4.Asegurarse de que las carpetas (Styles, Assets, secciones, etc.) estén en las rutas correctas.
